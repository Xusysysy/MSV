# MSV 乐谱查看器 v1.6

## 新增功能

- **横屏双页模式（Spread View）**：横屏时自动检测，当单页宽度不足屏幕一半时，左右并排显示两页，翻页以两页为单位
- **横屏渐变遮罩**：双页模式下左右边缘添加背景色渐变遮罩，柔化边缘过渡
- **页码全局记忆**：每个文件独立记忆阅读页码，冷启动和谱架切换均可恢复上次位置
- **分级渲染策略**：当前页同步优先渲染，±1 页全质量并行，±2\~3 页降分辨率渲染，±4\~5 页最低分辨率预加载
- **缓存自动清理**：渲染范围外的缓存页面和磁盘文件自动清理，避免内存溢出

## 性能优化

- 页面渲染改为 async 并行，加载速度大幅提升
- 手势处理从 coroutineScope + launch 改为 awaitEachGesture，消除首帧手势丢失
- 加载动画等待当前页 ±2 页全部就绪后才隐藏，确保首次交互无延迟
- 翻页动画改用 spring 弹簧动画，手感更流畅自然

## 修复

- 修复翻页动画结束后页面位置偏移闪回的问题（snapTo(0f) 与 onNextPage 执行顺序）
- 修复切换谱子后 displayW/flipUnit 闭包捕获旧值导致翻页目标位置错误
- 修复切换谱子后 transition 残留非零值导致居中位置偏移
- 修复缩略图面板打开后当前页未居中显示
- 修复加载 Overlay 半透明导致触摸穿透
- 移除页面阴影效果
